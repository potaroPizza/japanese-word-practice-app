# 일본어 단어 연습 앱 - MVP 기획서

## 개요

일본어 공부를 위한 단어 연습 게임 앱. 다양한 학습 모드를 통해 JLPT 단어를 효과적으로 암기할 수 있도록 돕는다.

## 대상 사용자

- 일본어를 공부하는 한국어 사용자
- JLPT N5~N1 수준의 단어 학습이 필요한 사람

## 기술 스택

| 항목 | 선택 | 사유 |
|------|------|------|
| 프레임워크 | Flutter Web (Dart) | 웹 앱으로 배포하여 iOS/Android 구분 없이 브라우저에서 접근 가능 |
| 배포 | GitHub Pages | 무료, git push만으로 자동 배포, 별도 가입 불필요 |
| 데이터 저장 | 브라우저 로컬 저장 (localStorage / IndexedDB) | 서버 불필요, 기기 내 학습 기록 유지 |
| 단어 데이터 | 앱 내장 | JLPT N5~N1 단어장을 앱에 포함 |

## 플랫폼 결정 배경

- 고객(누나)이 iPhone 사용자
- iOS 앱 빌드에는 Mac + Apple 개발자 계정($99/년) 필요
- Flutter Web으로 빌드하면 iPhone Safari에서 바로 접근 가능
- 홈화면 추가(PWA)로 네이티브 앱처럼 사용 가능

## 핵심 기능

### 0. 접속 인증

- 앱 접속 시 4자리 PIN 입력
- PIN은 SHA-256 해시로 저장 (소스에서 원본 확인 불가)
- 5회 실패 시 24시간 잠금

### 1. 플래시카드

- 일본어 단어를 보여주고 카드를 뒤집으면 한국어 뜻이 나옴
- 반대 방향(한국어 -> 일본어)도 지원
- "알고 있음" / "모름" 체크 가능

### 2. 객관식 퀴즈

- 일본어 단어를 보고 4지선다 중 정답 선택
- 정답/오답 즉시 피드백

### 3. 빈칸 채우기

- 한국어 뜻을 보고 일본어 단어를 직접 입력
- 히라가나/카타카나 입력 지원

### 4. 진도 관리

- 맞힌/틀린 기록 저장
- 틀린 단어 위주 복습 기능
- JLPT 레벨별 학습 진도 확인

## 단어 데이터

- JLPT N5 ~ N1 전체 레벨
- 각 단어 항목: 일본어(한자), 히라가나/카타카나 읽기, 한국어 뜻
- 웹서칭을 통해 공개된 JLPT 단어 목록 수집

## 데이터 저장

- 브라우저 로컬 저장 (localStorage / IndexedDB)
- 서버/로그인 없음 (MVP 범위)
- 같은 브라우저에서 접속하면 학습 기록 유지

## MVP 범위 외 (향후 고려)

- 서버 연동 및 사용자 계정
- 단어장 직접 등록/편집
- 음성 발음 재생
- 학습 통계 및 그래프
- 스페이스드 리피티션(간격 반복) 알고리즘
