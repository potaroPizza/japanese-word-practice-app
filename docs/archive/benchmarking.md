# 일본어 단어 학습 앱 벤치마킹

## 주요 앱 분석

### 1. Anki

- **개요:** 오픈소스 SRS(간격 반복) 기반 플래시카드 앱. 일본어 외 모든 분야 암기 학습에 사용
- **학습 방식:** 플래시카드 (자기 평가: Again/Hard/Good/Easy), 빈칸채우기(Cloze) 카드
- **강점:** SRS 알고리즘 효과 검증됨, 커스터마이징 자유도 최고, 커뮤니티 덱 방대, PC/Android 무료
- **약점:** UI가 투박하고 구식, 초보자 진입장벽 높음, iOS 유료($24.99)
- **가격:** PC/Android 무료 / iOS $24.99

### 2. Takoboto

- **개요:** Android 전용 일본어 사전 겸 학습 앱
- **학습 방식:** 플래시카드, 단어 리스트 브라우징, 사전 검색 중심
- **강점:** 오프라인 사전 우수, 검색 빠르고 정확, 한자 정보 상세
- **약점:** Android 전용, 학습 기능 부족, SRS 없음, 한국어 미지원
- **가격:** 무료(광고) / 유료 광고 제거

### 3. Obenkyo

- **개요:** Android 전용 일본어 종합 학습 앱 (히라가나/카타카나/한자/문법)
- **학습 방식:** 플래시카드, 객관식 퀴즈, 필기 연습(터치로 한자 쓰기), SRS 복습
- **강점:** 완전 무료, 초보~고급 포괄, 한자 획순 학습, 오프라인 사용
- **약점:** UI 디자인 오래됨, Android 전용, 음성 지원 부족
- **가격:** 완전 무료

### 4. Japanese by Renzo

- **개요:** iOS/Android 일본어 사전 및 학습 앱
- **학습 방식:** 플래시카드, 퀴즈, 텍스트 분석(문장 붙여넣기 시 단어별 분석)
- **강점:** 모던하고 깔끔한 UI, 사전+학습 잘 통합됨, 오프라인 사용
- **약점:** 프리미엄 비쌈, 한국어 미지원
- **가격:** 기본 무료 / 프리미엄 구독 월 $7~8

### 5. WaniKani

- **개요:** 한자 및 어휘 학습 특화 웹/앱 서비스. SRS + 니모닉(연상기억법)
- **학습 방식:** 타이핑 입력(읽기/뜻 직접 입력), 레슨 -> 리뷰 -> 번업 단계
- **강점:** 체계적 커리큘럼(급진 -> 한자 -> 어휘), 니모닉 효과적, 레벨업 게이미피케이션
- **약점:** 유료 구독제, 학습 순서 커스텀 불가, 초반 느림
- **가격:** 레벨 3까지 무료 / 월 $9 / 연 $89 / 평생 $299

### 6. Memrise

- **개요:** 다국어 학습 플랫폼. 원어민 영상 기반 학습이 특징
- **학습 방식:** 플래시카드(영상+텍스트), 객관식 퀴즈, 듣기 퀴즈, 타이핑 입력
- **강점:** 원어민 영상으로 실제 발음 학습, 모던한 UI, 게이미피케이션
- **약점:** 무료 버전 제한 큼, 커뮤니티 코스 품질 편차, 문법 설명 부족
- **가격:** 기본 무료 / Pro 월 $8.49 / 연 $59.99 / 평생 $199.99

### 7. Kanji Study

- **개요:** 한자 학습 특화 앱
- **학습 방식:** 플래시카드, 필기 연습, 퀴즈
- **강점:** 한자 학습 최고 평가, 획순 애니메이션, 깔끔한 UI
- **가격:** 기본 무료 / 프리미엄 약 $12

### 8. 한국 사용자 대상 앱

- **말해보카 일본어:** 단어 암기 + 음성 인식 발음 연습, 한국어 UI, 구독제
- **일본어 무작정 따라하기 (길벗):** 한국 교재 기반, 음성 듣기/따라 말하기
- **네이버 사전:** 일한/한일 사전, 예문, 단어장, 무료

## 우리 앱에 적용할 시사점

1. **SRS는 필수** — 거의 모든 성공적인 학습 앱이 SRS를 채택. MVP 범위 외로 빼뒀지만 우선순위 높게 고려
2. **한국어 지원 앱 부족** — 한국 사용자 대상 일본어 단어 앱 시장에 빈틈이 있음. 차별화 포인트
3. **다양한 학습 모드** — 플래시카드 단일 방식보다 퀴즈, 타이핑 등 복합 방식 선호. 이미 기획에 반영됨
4. **UI/UX가 경쟁력** — Anki는 기능 최고이나 UI 불만 많고, Renzo는 UI로 차별화 성공
5. **오프라인 지원** — 로컬 저장 방식이므로 자연스럽게 충족
6. **게이미피케이션** — WaniKani의 레벨업, Memrise의 포인트 시스템 등이 사용자 유지에 효과적
